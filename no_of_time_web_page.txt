1.Write a PHP script to keep track of number of times the web page has been accessed
(Use Session Tracking).

Solution:
<?php
if(!isset($_COOKIE['access'])) {
    $page_access = 1;
} else {
    $page_access = $_COOKIE['access'] + 1;
}

// Set updated cookie
setcookie('access', $page_access);

// Display count
echo "This page is accessed " . $page_access . " times";
?>

run:localhost/counter.php
